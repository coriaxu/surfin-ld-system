## 🔍 第二轮深度调研总结

### 行业顶级实践（麦肯锡/贝恩级别）

| 设计原则       | 具体应用                       |
| -------------- | ------------------------------ |
| **金字塔原则** | 先呈现核心结论，再展开支撑论据 |
| **数据可视化** | 用动态数字、图表展示成果       |
| **故事化叙事** | 从问题 → 方案 → 结果的完整旅程 |
| **品牌一致性** | 统一的视觉系统和设计语言       |
| **结构化标签** | 按主题/行业/场景分类，便于检索 |

### 卡片式布局最佳实践

| 要素         | 规范                                                  |
| ------------ | ----------------------------------------------------- |
| **视觉层次** | 标题 20px + 描述 14px，突出重点                       |
| **间距系统** | 基于 8px 栅格，保持呼吸感                             |
| **卡片结构** | Header（图标/图片）+ Body（标题+描述）+ Footer（CTA） |
| **交互状态** | hover 放大 + 阴影加深 + 边框高亮                      |
| **响应式**   | 自适应 2-3-4 列布局                                   |

### 高端培训平台设计趋势

| 趋势           | 说明                               |
| -------------- | ---------------------------------- |
| **极简留白**   | 减少认知负担，聚焦核心内容         |
| **毛玻璃效果** | Glassmorphism 增加层次感和高级感   |
| **微动效**     | hover 缩放、渐入渐出，增强交互反馈 |
| **品牌色彩**   | 用色彩区分不同模块，建立视觉识别   |
| **双语并行**   | 国际化训练营，中英双语展示专业度   |

## 🎨 方案 B 精细化设计

### 设计主题：「琥珀智库」

> **设计理念**：用温暖的琥珀金色调传递「经验沉淀」「智慧结晶」的寓意，与金融小百科的翠绿色形成和谐对比。

### 配色方案

| 变量名                 | 色值                                        | 用途        |
| ---------------------- | ------------------------------------------- | ----------- |
| `--case-primary`       | `#B8860B`                                   | 主色-暗金   |
| `--case-primary-light` | `#DAA520`                                   | 亮金-悬停态 |
| `--case-accent`        | `#FFF8DC`                                   | 浅金背景    |
| `--case-surface`       | `rgba(255, 248, 220, 0.6)`                  | 毛玻璃底色  |
| `--case-border`        | `rgba(184, 134, 11, 0.2)`                   | 边框色      |
| `--case-gradient`      | `linear-gradient(135deg, #FFF8DC, #FFEFD5)` | 背景渐变    |

### 交互设计细节

| 元素             | 交互效果                              |
| ---------------- | ------------------------------------- |
| **统计卡片**     | 数字计数动画（0→8+），悬停微缩放      |
| **案例卡片**     | hover: 上浮 8px + 阴影加深 + 边框高亮 |
| **Tab 切换**     | 下划线滑动动画，内容淡入淡出          |
| **国家手册卡片** | 国旗 emoji 放大 + 卡片边框闪烁金色    |
| **CTA 按钮**     | 渐变背景 + 悬停脉动光晕 + 图标右移    |

### 技术架构

| 模块         | 实现方式                          |
| ------------ | --------------------------------- |
| **页面容器** | `#page-caselib`，与其他 page 并列 |
| **导航入口** | `nav-item[data-page="caselib"]`   |
| **CSS 变量** | 专属 `--case-*` 变量，局部覆盖    |
| **Tab 组件** | 复用 AI 知识库的 `.ai-tabs` 模式  |
| **外链跳转** | `window.open(url, '_blank')`      |
| **数据存储** | 硬编码在 HTML 中（内容稳定）      |
